
Este projeto é uma Calculadora de IMC (Índice de Massa Corporal) desenvolvida em React.
O usuário informa seu peso e altura, e a aplicação calcula automaticamente o IMC, exibindo também uma tabela de classificação para interpretação do resultado.


Tecnologias utilizadas

React e Vite

JavaScript (ES6+)

CSS

Como rodar: 

git clone https://github.com/NathanaelZampier/calculadora_imc

npm install

npm run dev
